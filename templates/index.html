Info and instructions

This visualization uses datasets from IMDb which can be found at https://www.imdb.com/interfaces/. The code is available at https://github.com/kaisuliinos/internet-movie-visualizer.

The Top list includes titles with 1000 or more votes. If none of the selected titles have 1000 votes after filtering the data, all of the titles are included.

In Titles released per year, series are included in each year between their start and end years.

In most popular genres, click a bubble to see the highest-rated titles and the amount of released titles for a specific genre. You can select multiple genres by Shift+click. Undo the selection by clicking between the bubbles.
